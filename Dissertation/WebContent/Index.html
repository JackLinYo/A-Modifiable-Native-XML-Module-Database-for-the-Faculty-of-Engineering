A Modifiable Native Module Database for the Faculty of Engineering


Purpose

This project aim to provide a system, which allows the users to modify the data that from the Faculty of Engineering and store the changed data to the Native XML Database

Instruction

The upload page stands for allowing user to update or change the latest data which referred to a particular department. The users can fill up the form which provide a form that related to Faculty of Engineering, with the departments below to the Faculty of Engineering,module name, module number, credits of the module and extra information to the modules. After uploading the latest data, the users can search for the specific module. All the information will be displayed in the search page.
